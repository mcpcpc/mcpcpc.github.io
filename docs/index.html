xwm

xwm (“XCB Window Manager”) is a tiny floating window manager written using the XCB protocol. This project provides a limited set of out-of-box features, including parametric window settings, configurable and extendable key bindings and default utilities, as well as POSIX C99 and MISRA-C compliance.

Getting started is a easy as using git and make commands to build and install xwm:

git clone https://github.com/mcpcpc/xwm
cd xwm
make
sudo make install
